## Steps to Build Back-end API with Python Django

1. Install Python

2. Create folder 			: mkdir MovieRaterApi

3. Move to created fold			: cd MovieRaterApi

4. Setup venv(Virtual Environment) 	: python -m venv venv

5. Activate venv 			: source venv/Sripts/activate

6. Install Django 			: pip install Django

7. Install Djangorestframework 		: pip install Djangorestframework

8. Switch to IDE 			: PyCharm or VScode

9. Start Django Project 		: django-admin startproject movierater .

10. Start Django Application 		: django-admin startapp api

11. Make sure everything is workign 	: python manage.py runserver

12. Check link				: http://127.0.0.1:8000/

13. Set detault run mange.py if using 	: PyCharm IDE

14. Apply built in Migrations		: python magnage.py migrate

15. Add 'rest_framework', to INSTALLED_APPS in settings.py

16. Add created app(s) - 'api', to INSTALLED_APPS in settings.py

17. Add urls.py file to django app 'api' directory.

18. Copy "import and urlpatterns" from project url to created app url file.

19. Apply Migrations			: python magnage.py migrate

20. Create superuser			: python mange.py createsuperuser

21. Login to admin to see.

22. Set up "router" in app urls.py file as needed.

23. Create model classes in app models.py file as needed.

24. Register model classes in app admin.py file as needed.

25. Check admin portal to see registered models.

26. Create a new serializers.py file in app directory and set up serializer classes as needed.

27. Set up ViewSet for each serilizer and configure as needed.

28. Register ViewSet in our router in the app urls.py file.

29. Test api endpoint using Postman, if needed use browser to troubleshoot problems.

30. Test database, create records using Postman, and perform more test on POST, GET, PUT, PATCH, DELETE.

31. Create custom logic in the app view.py file e.g create only if does not exist, else update if exist.

32. Set up Token Auth, add 'rest_framework_authtoken' to INSTALLED_APPS in settings.py

33. run migration 				: python mange.py migrate

34. Generate token for current user.

35. Set up path for authentication in project url.py file.

36. Set up authorization

37. Register Users

38. Restrictions
